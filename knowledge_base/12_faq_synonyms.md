# Module 12: FAQ & Arabic-English Synonyms

## 1. Arab-English Synonyms Dictionary (قاموس المرادفات للذكاء الاصطناعي)
* **كشف / معاينة / فحص** = Consultation / Visit / Examination
* **روشتة / علاج / وصفة** = Prescription / Medications sheet
* **فاتورة / حساب / إيصال** = Invoice / Bill / Receipt
* **موعد / حجز** = Appointment
* **مخزن / مستودع / صيدلية** = Inventory / Stock / Pharmacy
* **خزنة / مالية / درج** = Treasury / Cash drawer

## 2. End-User FAQ Catalog (الأسئلة الشائعة)
* **س: كيف أقوم بإضافة مريض لا يملك رقم هاتف (مريض بدون واتساب)؟**
  * *ج: يمكنك ترك خانة رقم الهاتف فارغة وسيقوم النظام تلقائياً بإنشاء معرّف محلي متسلسل وفريد له لتفادي أخطاء قواعد البيانات.*
* **س: لماذا لا تصل رسائل الواتساب التلقائية لبعض الأرقام؟**
  * *ج: يرجى التأكد من اختيار الكود الدولي الصحيح أولاً (مثل +20 لمصر، +966 للسعودية) قبل كتابة رقم الهاتف. الأرقام بدون كود دولي صحيح يتم رفضها.*
* **س: كيف أعرف أن صنف دواء معين قارب على الانتهاء من المخازن؟**
  * *ج: سيظهر تنبيه أحمر تلقائي في أعلى الشاشة يوضح انخفاض كمية الصنف بمجرد هبوطها عن الحد الحرج المحدد بالجرام أو العبوة.*
